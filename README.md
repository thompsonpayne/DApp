# Logistics App

This is a learning project for practicing C++ desktop application development.

## Tech stack

- C++23
- Qt 6 (Widgets, Sql)
- CMake
- SQLite
- nlohmann/json

## Build

```bash
cmake -S . -B build
cmake --build build
```

## Run

```bash
./build/app
```

macOS app bundle (if generated):

```bash
./build/app.app/Contents/MacOS/app
```
